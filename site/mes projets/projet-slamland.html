  • Outils: Eclipse IDE

Contexte du projet

Réalisation d'une application java qui consiste a la gestion d'un parc d'attraction, l'application doit permettre d'ajouter/supprimer des visiteurs, attraction et commerce, affichage d'une list des visiteurs, attraction et commerce au format d'un tableau, et de calculer le chiffre d'affaire du parc d'attraction.
Une documentation en java et des tests uniataire ont été réaliser tout le long du projet.


Synthése

Le projet a été mené jusqu'au bout, j'ai eu quelque difficulté a réaliser ce projet a cause de nombreux bug causé par le java swing (IHM).
